
# Система управления банковскими картами

Backend-приложение на Java (Spring Boot) для управления банковскими картами. Позволяет:
- Создавать и управлять картами
- Просматривать карты
- Выполнять переводы между своими картами

Запуск: `./mvnw spring-boot:run`
Документация API: `docs/openapi.yaml`
Тесты: Юнит-тесты в папке test.
